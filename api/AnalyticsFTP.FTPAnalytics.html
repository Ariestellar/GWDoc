    Class FTPAnalytics

    Класс необходимый для сбора аналитики (путем обработки событий) и записи ее в файлики а так же инициирования отправления на сервер
    Находится на объекте AnalyticsManger в сценах тренажеров

    для сбора статистики и последующей записи необходимо расставить события в соответсвующие названиям точках:
    -Пользователь начал прохождение режима симулятора
    -Пользователь начал прохождение шага
    -Пользователь допустил ошибку
    -Пользователь завершил прохождение шага
    -Пользователь завершил прохождение режима симулятора
    Зависимости:
    Для работы необходимы кастомные статические классы:
    -EventManager
    -FTPManager
    -GlobalTrainerData
    -GlobalUserData
